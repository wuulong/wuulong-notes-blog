# 夫妻樹 (Couple Tree)

后里馬場旁的百年神木，象徵永恆的愛情。

## 景點介紹
后里夫妻樹（又稱日月神木）由一棵千年樟樹與一棵兩百年榕樹緊密相依而成。當地下流傳著淒美的愛情傳說，也讓這裡成為情侶騎車必訪的打卡點。樹下設有休憩廣場，是挑戰九號隧道前的重要補給站。

## 傳說故事
相傳早期有一對戀人因家人反對而在此殉情，後化身為相依偎的兩棵大樹，世世守護這片土地。

## 參觀資訊
*   **位置**: 后里馬場旁，九號隧道入口前。
*   **設施**: 洗手間、休息石椅。
