# 梅子鐵橋 (Meizi Iron Bridge)

跨越大甲溪的第二座鐵橋。

## 景點介紹
位於東勢區與石岡區交界，是東豐自行車綠廊進入東勢前的最後一道關卡。這座鐵橋原為東勢線鐵路橋，跨越寬闊的大甲溪河床。與花樑鋼橋不同的是，結構較為簡潔，視野更加開闊。

## 歷史災難與重建
*   **2006水災**: 豪雨沖斷了部分橋墩，導致橋面斷裂。
*   **修復**: 後經重建，改為目前所見的鋼拱橋與水泥橋墩複合形式，安全性大幅提升。
