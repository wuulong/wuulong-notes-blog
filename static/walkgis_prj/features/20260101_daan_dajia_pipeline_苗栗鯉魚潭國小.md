# 苗栗鯉魚潭國小

座落於鯉魚潭水庫旁，是推動水資源與生態教育的**示範基地**。

## 環境教育聯盟
*   **看見台灣**：加入「看見．齊柏林基金會」校園環境教育計畫，引導學生以鳥瞰視角守護家鄉水資源。
*   **水庫環教**：積極參與中區水資分署推動的鯉魚潭水庫環教課程，利用地利之便，將水庫變成最生動的自然教室。
