# 枕頭山 (苗栗)

苗栗三義鄉的特殊地標，因山形似枕頭而得名。

## 關鍵隧道工程
鯉魚潭第二原水管在此處進行艱鉅的隧道工程。
*   **長度**：隧道長約 1.5 公里 (另一說為 1,854 公尺)。
*   **挑戰**：工程需穿越複雜地質，但已順利貫通，讓巨大原水管能穿山而過，將水送往大安溪南岸。
