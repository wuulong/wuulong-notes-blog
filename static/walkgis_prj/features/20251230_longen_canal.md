# 隆恩圳 (東大路旁)

穿越三百年時光的城市藍帶，新竹三大古圳之首。

## 景點介紹

隆恩圳開鑿於清康熙年間（約1715年），由王世傑家族開墾，是新竹地區歷史最悠久的水圳，早於竹塹城的建立。

*   **歷史地位**: 原名「四百甲圳」，後因歸公有而改名「隆恩圳」。它見證了新竹從康熙年間至今的三百年開發史。
*   **城市綠帶**: 位於東大路旁的隆恩圳，經過新竹市政府的「隆恩圳景觀空間串連計畫」，已從封閉的水溝變身為親水的城市綠帶。
*   **設計巧思**: 改造後的隆恩圳保留了古圳的水道紋理，並加入了高架綠地、散步道與遊戲場，讓市民能親近這條流淌三百年的活歷史。
