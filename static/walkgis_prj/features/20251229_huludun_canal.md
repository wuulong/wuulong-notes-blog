# 葫蘆墩圳 (Huludun Canal)

台中盆地的農業之母，張達京的開墾傳奇。

## 歷史背景
清雍正年間，漢人通事張達京與平埔族岸裡社合作，以「割地換水」的創新模式開鑿水圳。這是台中地區最早、規模最大的水利工程，奠定了台中繁榮的農業基礎。

## 文化意義
*   **取水口**: 位於豐原萬順二街盡頭，近石岡水壩。
*   **張達京傳奇**: 見證了漢人與原住民的合作開發史。
*   **水岸花都**: 豐原市區段近年進行掀蓋整治，成為美麗的親水公園。
