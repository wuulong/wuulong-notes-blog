# 街頭圳幹線

橫山鄉的田園動脈，滋養田寮村的優質水源。

## 景點介紹

街頭圳幹線位於新竹縣橫山鄉田寮村，其取水口位於上坪溪堤防旁。這條水圳是橫山鄉農業發展的重要支柱。

*   **灌溉核心**: 引上坪溪清澈水源，灌溉田寮村三分之一的土地及下游橫山村的水稻田，受益面積達 50 公頃。
*   **農村風貌**: 沿著水圳漫步，可以看見典型的客家農村景觀：水稻田、菜園與散落的農舍。
*   **設施改善**: 近年新竹農田水利會投入經費修繕圳路，確保了輸水效率，也讓這條古老水圳能持續服務新一代的農民。
