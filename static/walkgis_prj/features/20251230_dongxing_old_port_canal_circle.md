# 東興舊港圳圓環

全台罕見的「水利雙圓環」，日治時期的分水智慧。

## 景點介紹

位於竹北的東興舊港圳圓環，是一個極具特色的水利設施。在日治時期隘口堤防興建後，為了解決東興圳與舊港圳的配水問題，工程師設計了這個獨特的圓環分水工。

*   **分水原理**: 水流進入圓形水池後，利用圓環的緩衝與導流作用，將水流精確且公平地分配到左側的東興圳與右側的舊港圳。這展示了流體力學在早期水利工程中的巧妙應用。
*   **歷史意義**: 東興圳（乾隆年間）與舊港圳都是竹北重要的灌溉命脈。這個圓環不僅解決了紛爭，也成為當地獨特的地景。
*   **現代價值**: 雖歷經時代變遷，這個水利圓環至今仍維持運作，見證了竹北從農業聚落轉型為現代城市的歷程。
