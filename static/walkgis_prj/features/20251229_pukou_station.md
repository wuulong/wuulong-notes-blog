# 朴口車站 (Pukou Station)

舊東勢線的簡易招呼站遺跡。

## 景點介紹
朴口車站原是台鐵東勢線上的一個無人招呼站，主要服務當地的通勤學生。廢線後，月台被完整保留下來，並重建了充滿懷舊感的候車亭。騎車經過時，不妨停下來感受一下昔日單純的歲月。

## 歷史小檔案
*   **啟用**: 1961年。
*   **廢止**: 1991年。
*   **現況**: 東豐綠廊上的歷史地標。
