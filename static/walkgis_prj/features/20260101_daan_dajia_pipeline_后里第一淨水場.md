# 后里第一淨水場

位於中科后里園區內，是本計畫中新建的關鍵淨水設施，預計 2026 年完工。

## 核心功能
*   **系統串接**：地理位置上連接了北側的「鯉魚潭系統」與南側的「大甲溪系統」，實現水源的互通有無。
*   **處理能力**：設計每日處理量達 20 萬噸，主要供應中科園區及大台中地區。
*   **綠能設施**：場內規劃設置小水力發電設備，利用水流位能發電。
