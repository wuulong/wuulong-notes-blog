# 嘉南大圳濁幹線調蓄池

- **類別**: 水利設施-雲林管理處
- **地址**: 632台灣雲林縣虎尾鎮工專路281號
- **評分**: 0

## 簡介
為解決濁水溪枯水期缺水及高濁度問題，於濁幹線沿線設置的調蓄池（人工湖），夜間儲水、日間供水。

## 歷史與故事
近年來推動的現代化水利設施，提升用水韌性。

## 特色
具備沉砂、蓄水功能，水域寬廣平靜。

- **功能**: 蓄豐濟枯
