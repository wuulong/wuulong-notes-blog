# 大安溪水管橋 (后里圳水橋)

此處是跨越大安溪的關鍵節點，見證了跨越百年的水利工程對話。

## 新舊交映
*   **古蹟**：原有的「后里圳水橋」建於1913年，是優美的磚造拱形渡槽。
*   **新工程**：大安大甲溪聯通管計畫在此新建一座現代化的水管橋，長度約 0.85 公里，讓鯉魚潭第二原水管能跨越寬闊的大安溪河床。
