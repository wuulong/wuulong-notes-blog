# 后里圳 (示範電廠)

后里圳始建於1909年，是開發后里台地農業的重要命脈。

## 低落差示範電廠
利用后里圳的水位落差，這裡設置了一座小水力示範電廠。這代表了水資源的多元利用：不僅灌溉農田，還能產生綠色能源，是大安大甲溪水資源運用的一個精采縮影。
