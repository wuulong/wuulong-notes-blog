# 東勢本圳 (Dongshi Main Canal)

滋潤東勢山城的百年水脈。

## 歷史背景
開鑿於清嘉慶年間（1799年），由地方仕紳合力開鑿。引大甲溪水灌溉東勢地區農田，與老圳、大茅埔圳並列為當地重要水利系統。

## 參觀資訊
*   **取水口**: 位於大茅埔 (近大甲溪)。
*   **景觀**: 沿途流經東勢市區與農田，是觀察傳統農村水利的好去處。
