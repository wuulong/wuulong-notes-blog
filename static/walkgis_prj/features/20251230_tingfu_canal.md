# 汀甫圳

新竹市最長的水圳綠廊，承載著「口琴橋」的青春記憶。

## 景點介紹

汀甫圳始建於清朝嘉慶年問，原名「泉興圳」，後由何汀甫接手管理並改名「何勝圳」。1924年水圳公有化時，何汀甫大方捐出水圳，政府為感念其貢獻，將其定名為 **「汀甫圳」**。

*   **城市綠脈**: 總長約 15.43 公里，引頭前溪水穿越新竹精華區（如交大博愛校區、赤土崎），是新竹市早期的生命動脈。
*   **口琴橋傳奇**: 沿線最著名的地標是橫跨客雅溪的「口琴橋」（渡槽），因其特殊的鋼筋混凝土格狀結構形似口琴而得名。這裡曾是許多新竹學子的約會聖地，承載著濃濃的在地記憶。
*   **現代轉身**: 雖然部分渠段隨都市化轉為排水功能，但經過近年的景觀整治，汀甫圳已變身為市民休閒的親水綠廊，連結了歷史與現代生活。
