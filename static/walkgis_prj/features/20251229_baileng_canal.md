# 白冷圳 (Baileng Canal)

翻山越嶺的生命之水，倒虹吸管的工程奇蹟。

## 歷史背景
日治時期為了灌溉新社台地的大南庄蔗苗養成所，工程師磯田謙雄設計了這條引水工程。水源取自大甲溪上游白冷高地，全長 16.6 公里，全憑重力輸送，不需動力。

## 工程亮點
*   **倒虹吸管**: 利用連通管原理，讓水流克服高低落差翻越山谷。不遠處可見新舊兩座巨大的綠色鋼管並列，其中 2 號倒虹吸管落差達 90 公尺，極為壯觀。
*   **位置**: 白冷圳紀念公園 (台中市新社區)。
*   **歷史建築**: 入選台灣歷史建築百景。
