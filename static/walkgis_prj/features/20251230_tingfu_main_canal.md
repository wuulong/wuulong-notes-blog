# 汀甫圳幹線

穿越學府路的綠色散步道，都市裡的水利美學。

## 景點介紹

汀甫圳幹線在流經新竹市東區（特別是學府路、交大博愛校區一帶）的路段，近期進行了顯著的景觀改善工程，成為「汀甫圳通學步道」的一部分。

*   **景觀縫合**: 工程結合了砌卵石護岸與仿木欄杆，保留了水圳的古樸質感，同時提升了步行的安全性與舒適度。
*   **生態棲地**: 改善後的渠道不僅排水順暢，也為城市中的微生態提供了棲息空間，常見白鷺鷥與夜鷺在此覓食。
*   **生活連結**: 這段水圳串聯了校園與社區，讓匆忙的都市人能停下腳步，聽見流水的聲音，是都市水利工程「景觀化」的成功案例。
