# 後庄埤親水公園

- **類別**: 水利設施-雲林管理處
- **地址**: 640台灣雲林縣斗六市大學路三段
- **評分**: 4.1

## 簡介
原為灌溉用的後庄埤，經整治後轉型為親水公園。園區內的「天空之島」造景與蜿蜒水岸，成為斗六市民的新休憩點。

## 歷史與故事
早期為單純農用埤塘，隨都市發展曾一度荒廢，2024年完成改造。

## 特色
兼具灌溉、滯洪與觀光休閒功能。

- **位置**: 雲林縣斗六市
