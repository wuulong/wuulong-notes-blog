# 三叉埤

位於橫山鄉的隱世秘境，百年埤塘孕育豐富生態。

## 景點介紹

三叉埤位於新竹縣橫山鄉，約在清朝嘉慶年間（約1860年）由當地陳姓人家以人工挑土築堤而成，因匯集三座山谷的溪流而得名。這座百年水利設施至今仍保留許多自然風貌，被譽為橫山鄉的「秘境」。

*   **歷史價值**: 早期灌溉百公頃良田，見證了先民開墾的艱辛。埤塘邊有一座古樸的石板伯公廟和兩棵被保護的老樟樹，充滿人文氣息。
*   **生態與觀光**: 環湖步道生態豐富，夏季蓮花盛開，是賞鳥與攝影的熱門地點。曾是客家文學大戲《寒夜續曲》的拍攝地，增添了幾分文學浪漫。
*   **重生與保育**: 近年政府補助進行生態塊護岸工程，並由當地社造團體維護，設置人力小水車等設施，使其成為兼具歷史教育與生態保育的優質景點。
