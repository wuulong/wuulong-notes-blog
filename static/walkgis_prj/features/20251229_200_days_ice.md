# 200days冰店 (200days Ice)

東豐綠廊最美的純白玻璃屋。

## 景點介紹
這是一間專賣古早味芋仔冰的網美店。純白色的三角形玻璃建築，在綠意盎然的自行車道上格外顯眼。店名由來是因為芋頭成熟期約需 200 天，以此象徵對食材的堅持。

## 招牌必點
*   **芋仔冰**: 濃郁綿密，經典古早味。
*   **花生冰棒**: 香氣十足，還吃得到顆粒。

## 參觀資訊
*   **位置**: 東豐綠廊 1.1K 處。
*   **設施**: 冷氣開放、戶外座位區。
