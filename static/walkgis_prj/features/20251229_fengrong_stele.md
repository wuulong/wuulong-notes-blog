# 豐榮水利碑 (Fengrong Stele)

見證台中農業水利發展的歷史碑碣。

## 景點介紹
位於葫蘆墩圳取水口附近，建於 1939 年（昭和14年）。這座碑是為了紀念當時的水利工程整修而立，見證了豐原地區（葫蘆墩）的水利發展史。

## 參觀亮點
*   **歷史建築**: 2004年公告為歷史建築。
*   **展望平台**: 旁有觀景台可遠眺石岡水壩與大甲溪，視野遼闊。
