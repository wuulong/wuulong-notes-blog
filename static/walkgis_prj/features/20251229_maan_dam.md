# 馬鞍壩 (Ma'an Dam)

友善生態的綠色水壩，魚兒回家的路。

## 生態亮點
*   **座標**: 24.1844° N, 120.9133° E
*   **生態魚道**: 這是台灣首座設計有大型生態魚道的水壩，分為「水池式」與「丹尼爾式」，幫助溪流中的魚類洄游繁衍，展現了工程與生態共存的理念。
*   **位置**: 位於大甲溪下游，鄰近東勢與和平交界。
