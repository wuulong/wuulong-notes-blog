# 德基水庫 (Techi Reservoir)

台灣最高的雙曲線薄型拱壩，大甲溪水力發電的源頭。

## 工程與功能
*   **座標**: 24.2541° N, 121.1652° E
*   **台灣最高壩**: 壩高 180 公尺，也是台灣海拔最高的水庫（必達溪）。
*   **建築特色**: 雙曲線薄型拱壩設計，造型優美且結構強韌。
*   **功能**: 發電、防洪、給水。它是大甲溪發電廠系列的第一關，尾水排入下游繼續發電，將水資源利用到極致。
