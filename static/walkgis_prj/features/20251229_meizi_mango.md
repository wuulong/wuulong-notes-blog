# 梅子百年芒果樹 (Meizi Century-old Mango Tree)

守護梅子社區的三百歲老樹王。

## 景點介紹
位於東豐綠廊旁的小徑內（有指標），這棵芒果樹樹齡估計超過 300 年，樹幹粗壯需多人合抱。當地人稱之為「老芒果」，樹下供奉著土地公，是梅子社區的信仰中心與乘涼好去處。

## 奇聞軼事
*   **靈樹傳說**: 據說老樹具有靈性，保佑著村莊風調雨順。
*   **芒果產量**: 雖然年事已高，但每年夏天依然結滿芒果（土芒果），生命力旺盛。
