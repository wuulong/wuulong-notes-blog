# 情人木橋 (Lovers Bridge)

遠東跨距最長的景觀膠合木橋。

## 景點介紹
橫跨食水嵙溪的情人木橋，全長 40 公尺，最神奇之處在於 **全橋沒有橋墩**，僅由巨大的膠合木樑支撐。這是台灣引進北美技術的經典之作，具備高抗震能力。

## 周邊景點
*   **石岡土牛客家文化館**: 展示客家傳統建築與生活。
*   **大水車**: 全國最大的水車之一，位於橋下公園。
*   **休閒農業區**: 附近有許多花園與蜜蜂生態館，適合親子同遊。
