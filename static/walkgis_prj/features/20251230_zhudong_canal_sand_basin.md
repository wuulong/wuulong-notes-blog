# 竹東圳 (分水沉沙池)

守護寶二水庫的隱形功臣，運用重力沈澱泥沙的智慧。

## 景點介紹

竹東圳是新竹開拓史上規模最大的水利工程，建於日治時期（1928年），全長21公里，穿山越嶺將上坪溪水引入大新竹地區。而這裡的「分水沉沙池」則是整個系統的守門員。

*   **沈沙機制**: 在溪水注入寶山第二水庫之前，必須先流經此處。利用流速減緩的原理，讓水中夾帶的泥沙自然沉降，避免水庫淤積，確保水庫的壽命與水質。
*   **樞紐地位**: 這裡是竹東圳與寶山水庫系統的交會點，既供應了竹科學園區的工業用水，也滋潤了下游的農田。
*   **工程奇蹟**: 竹東圳沿線包含13座隧道與多座水路橋，展現了當年工程師克服地形障礙的精湛技術，沉沙池則是其中具代表性的功能性設施。
