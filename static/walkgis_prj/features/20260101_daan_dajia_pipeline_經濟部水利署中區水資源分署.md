# 經濟部水利署中區水資源分署

本計畫的執行機關，更是推動**公民參與**與**公私協力**的核心推手。

## 公民溝通
*   **教育推廣**：主動走入校園（如后里泰安國小、鯉魚潭國小），舉辦水資源環境教育，向下扎根。
*   **地方共好**：在推動重大工程（如聯通管計畫）的同時，積極與地方里民溝通，感謝地方支持，並致力於降低工程對社區的影響，展現了公共工程與在地共存的努力。
