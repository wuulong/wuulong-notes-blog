# 貓兒錠幹線3支線

竹北濱海的稻浪守護者，尋找「貓兒碇」的歷史足跡。

## 景點介紹

位於竹北市西區的貓兒錠幹線，服務著大義、尚義、崇義等濱海里別的廣大農田。

*   **地名由來**: 「貓兒錠」之名源於清雍正年間，漢人開墾挖井時挖出一柄由「貓兒木」製成的古船碇，充滿了傳奇色彩。
*   **農業地景**: 在鳳岡路一帶，貓兒錠幹線3支線穿梭於水稻田之間。每當風起，綠色稻浪與水圳波光相映，是竹北難得的純樸農村風光。
*   **水利網絡**: 這條支線是大新竹灌溉系統末稍的重要微血管，確保了濱海地區的農田也能獲得穩定的水源滋潤。
