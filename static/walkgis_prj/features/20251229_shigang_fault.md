# 石岡斷層月台 (Shigang Fault Platform)

被地震撕裂的月台，大自然的震撼教育。

## 景點介紹
位於東豐綠廊上，原為舊東勢線石岡車站月台。1999 年 921 大地震時，車籠埔斷層正好切過此處，造成月台與鐵軌嚴重扭曲變形，產生近 5 公尺的高低落差。保留至今的斷層遺跡，是極具教育意義的地質教室。

## 參觀資訊
*   **位置**: 鄰近 0 蛋月台。
*   **特色**: 隆起的鐵軌與斷裂的月台結構。
